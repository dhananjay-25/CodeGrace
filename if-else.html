If-Else Statements

What are Conditional If-Else Statements?

Conditional statements are an essential part of any programming language, including C. They help your program make decisions by evaluating expressions and executing specific blocks of code based on whether the conditions are true or false. The most basic conditional statement in C is the if-else statement.

The if Statement
The 'if' statement is used to execute a block of code only if a specified condition evaluates to true. Here's the basic syntax:

                
                    if (condition) {
                        // Code to be executed if the condition is true
                    }

If the condition inside the parentheses is true, the code inside the curly braces will be executed. Otherwise, the code block will be skipped, and the program will continue with the next statements

The else Statement
Sometimes, you want to execute a different block of code when the condition is false. That's where the 'else' statement comes into play. The 'else' statement is optional and follows the 'if' block. Here's the syntax: 

                    
                        if (condition) {
                            // Code to be executed if the condition is true
                        } else {
                            // Code to be executed if the condition is false
                        }

If the 'condition' is true, the code inside the 'if' block will run, and the code inside the else block will be skipped. Conversely, if the condition is false, the code inside the else block will execute.

Examples
Let's look at a couple of examples to understand how if-else statements work:

Example 1: Checking if a number is even or odd 

                        
                            #include <stdio.h>

                                int main() {
                                    int num;
                                    printf("Enter a number: ");
                                    scanf("%d", &num);
                                
                                    if (num % 2 == 0) {
                                        printf("%d is an even number.\n", num);
                                    } else {
                                        printf("%d is an odd number.\n", num);
                                    }
                                return 0;
                                }

Example 2: Determining if a person is eligible to vote 

                            
                                #include <stdio.h>

                                    int main() {
                                        int age;
                                        printf("Enter your age: ");
                                        scanf("%d", &age);
                                    
                                        if (age >= 18) {
                                            printf("Congratulations! You are eligible to vote.\n");
                                        } else {
                                            printf("Sorry, you must be at least 18 years old to vote.\n");
                                        }
                                    return 0;
                                    }

In the first example, the program checks if the entered number is even or odd. In the second example, it determines whether a person is eligible to vote based on their age.

Conditional if-else statements are the backbone of decision-making in C programming. As you become more proficient in coding, you'll find yourself using these statements frequently to control the flow of your programs. Practice writing different if-else conditions to master this concept!
