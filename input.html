Scanf Statement

Welcome to our C tutorials page! In this section, we'll cover the essential aspects of the scanf() statement in C, a fundamental function used for input. Understanding how scanf() works will enable you to receive user input and make your C programs more interactive and versatile.

What is 'scanf()'?

scanf() is a C library function that allows you to read formatted data from the standard input stream (usually the keyboard) and store it in variables. It is part of the stdio.h library, and its prototype looks like this:

            
              int scanf(const char* format, ...);

Here, the 'format' argument specifies the format of the input to be read, and the ellipsis '(...)' indicates that 'scanf()'' can take multiple arguments, matching the format specifiers in the format string.

Format Specifiers

In the format string, you use format specifiers to indicate the type of data you expect to read. Common format specifiers include:

  • %d: Integer
  • %f: Float
  • %lf: Double
  • %c: Character
  • %s: String
  • %x or %X: Hexadecimal intege
  • %u: Unsigned integer

Basic scanf() Usage

Here's a simple example of how to use scanf() to read an integer from the user: 


              #include <stdio.h>
          
              int main() {
                int num;
                printf("Enter an integer: ");
                scanf("%d", &num);

                printf("You entered: %d\n", num);
                return 0;
              }

In this code, we prompt the user to enter an integer, use %d as the format specifier to read the input, and store it in the variable num. The & operator is used to get the address of num, as scanf() requires a pointer to the variable it reads into.

Handling Multiple Inputs

You can use multiple format specifiers in a single scanf() call to read multiple inputs at once. Separate the format specifiers with spaces or newlines. For example: 

              
                #include <stdio.h>
            
                int main() {
                  int age;
                  float height;
              
                  printf("Enter your age and height (separated by a space): ");
                  scanf("%d %f", &age, &height);
              
                  printf("Age: %d, Height: %.2f\n", age, height);
                  return 0;
                }

Strings with scanf()
When reading strings with scanf(), be careful about buffer overflows. Use the %s format specifier and provide a buffer size to prevent potential issues

                
                  #include <stdio.h>
              
                  int main() {
                    char name[50];

                    printf("Enter your name: ");
                    scanf("%49s", name); // Limit the input to 49 characters to leave room for the null terminator.
                
                    printf("Hello, %s!\n", name);
                    return 0;
                  }
